#plugins

[在 Obsidian 中打开插件](obsidian://show-plugin?id=obsidian-style-settings)

## 概述

Style Settings 允许你自定义 Minimal 的颜色、字体大小、字体样式以及更多细节。虽然 [[Minimal Theme Settings]] 插件对于 [[快捷键]] 和基本自定义是必需的，但 Style Settings 插件为 Obsidian 界面的所有细节提供了更深层次的自定义。

为了更方便地查看更改，可以使用命令面板（默认 `CMD + P`）在新面板中打开 Style Settings。

## 导入和导出设置

Style Settings 提供了通过 Style Settings 面板右上角的 **导入** 和 **导出** 按钮来分享自定义设置的选项。

请注意，使用 **导出** 功能时，你修改过的所有主题的 Style Settings 偏好设置都会被导出。你可能需要编辑输出内容，仅包含你想要分享的样式。

你可以在 [Obsidian 论坛帖子](https://forum.obsidian.md/t/share-your-minimal-theme-screenshots-configuration/8983) 或官方 [Obsidian Discord](https://discord.gg/veuWUTm) 上的 [Minimal 频道](https://discord.com/channels/686053708261228577/931008597557649410) 中找到和分享 Minimal 配色方案。
