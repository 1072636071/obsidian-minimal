#contributing

要设置本地开发环境，请按照以下说明操作。Minimal 使用 [Grunt](https://gruntjs.com/) 运行一系列编译任务，并监视工作文件在发生更改时自动运行此序列。

欢迎通过 Pull Request 贡献。你可以帮助 [[插件支持]]、Bug 修复和功能改进。你也可以在 [Minimal](https://github.com/kepano/obsidian-minimal/issues?q=is%3Aopen+is%3Aissue+label%3A%22help+wanted%22) 和 [Minimal Theme Settings](https://github.com/kepano/obsidian-minimal-settings/issues?q=is%3Aissue+is%3Aopen+label%3A%22help+wanted%22) 仓库中找到 _help wanted_ 标签的问题。

## 设置

安装 Sass 和 Grunt

```
sudo gem install sass
npm install -g grunt-cli
npm install
```

## 定义本地路径

要将主题直接构建到你的 Obsidian 仓库中，将 `.env.example` 重命名为 `.env` 并将 `OBSIDIAN_PATH` 更新为你的 Obsidian 主题文件夹的本地路径。

## 运行

```
npx grunt
```

这会构建两个文件：`obsidian.css` 分发文件（用于社区主题商店），它是压缩版本，也会复制到你的仓库以实现实时重载；以及 `Minimal.css` 文件，它是未压缩的副本，保存在项目根目录中。
