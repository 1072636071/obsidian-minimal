#contributing

插件作者和开发者可以通过向 [GitHub 上的 Minimal 仓库](https://github.com/kepano/obsidian-minimal)提交 Pull Request 来添加 Minimal 主题的支持。

[src/scss/plugins](https://github.com/kepano/obsidian-minimal/tree/master/src/scss/plugins) 目录提供了当前支持的插件的 [[CSS 代码片段]] 示例。参见 [[本地开发]] 说明。
