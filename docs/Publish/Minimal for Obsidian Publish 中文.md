## 概述

Minimal Publish 是 Minimal 针对 [Obsidian Publish](https://obsidian.md/publish) 的精简版本。它遵循相同的设计原则，但针对 Web 使用进行了优化。该主题旨在为你的所有访问者提供快速加载体验。[查看 GitHub 仓库。](https://github.com/kepano/obsidian-minimal-publish)

## 功能

- 遵循 Obsidian Publish 主题的[最佳实践](https://docs.obsidian.md/Themes/Obsidian+Publish+themes/Best+practices+for+Publish+themes)
- 文件大小更小，约 **16KB**，而应用版本约 **247KB**，下载速度更快
- 提供多种[[配色方案]]
- 兼容[[辅助类]]，如[[卡片]]、[[图片网格]]、[[图片缩放]]和[[表格|表格样式]]
- 支持创建[[为 Obsidian Publish 创建图片画廊|可缩放的图片画廊]]

## 如何安装

1. 使用下面的链接下载主题，或前往 [GitHub 仓库](https://github.com/kepano/obsidian-minimal-publish)
2. 将 `publish.css` 文件复制到你的仓库根目录
3. 发布 `publish.css` 文件
