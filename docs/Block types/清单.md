## 复选框样式

使用 [[Style Settings]] 编辑复选框的形状和颜色

## 替代复选框

Minimal 支持多种替代复选框类型。这些允许你标记未完成、已取消、已重新安排等任务。

[[配色方案|配色方案]] 颜色应用于替代复选框，可以使用 [[Style Settings]] 中的扩展调色板进行自定义。

![[alt-checkboxes.png#interface]]

## 复选框图标

以下复选框样式可用

| 语法    | 说明   |
| ------- | ------ |
| `- [ ]` | 待办   |
| `- [/]` | 未完成 |
| `- [x]` | 已完成 |
| `- [-]` | 已取消 |
| `- [>]` | 已转发 |
| `- [<]` | 已排期 |
| `- [?]` | 问题   |
| `- [!]` | 重要   |
| `- [*]` | 星标   |
| `- ["]` | 引用   |
| `- [l]` | 位置   |
| `- [b]` | 书签   |
| `- [i]` | 信息   |
| `- [S]` | 储蓄   |
| `- [I]` | 想法   |
| `- [p]` | 优点   |
| `- [c]` | 缺点   |
| `- [f]` | 火焰   |
| `- [k]` | 关键   |
| `- [w]` | 胜利   |
| `- [u]` | 向上   |
| `- [d]` | 向下   |
