#blocks

嵌入，也称为[内容包含](https://en.wikipedia.org/wiki/Transclusion)，允许你在另一个笔记的文本流中包含一个笔记或块。参见[官方 Obsidian 文档](https://help.obsidian.md/How+to/Embed+files)了解其工作原理。

## 严格嵌入

默认情况下，嵌入通过边框在视觉上标识。但是，你也可以启用"严格嵌入"，它会移除样式，使内容包含无缝地出现在文本流中。

使用 [[Style Settings]] 插件全局启用严格嵌入。

## 辅助类

使用 [[辅助类]] 在笔记级别控制嵌入样式。

| 类名              | 说明                         |
| ----------------- | ---------------------------- |
| `embed-strict`    | 内容包含无缝地出现在文本流中 |
| `embed-underline` | 包含的文本添加下划线         |
