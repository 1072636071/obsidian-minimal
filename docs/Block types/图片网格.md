## 概述

图片网格是一种将 [[图片]] 排列为列的简单方法。此功能兼容 [[块宽度]] 的图片宽度选项。

图片网格可以通过 [[Minimal Theme Settings]] 和 [[快捷键]] 全局激活，或使用 [[辅助类]] 在逐文件基础上激活。

![[minimal-img-grid.png]]

## 演示

图片网格使用简单，不依赖任何特殊语法。你只需在图片之间添加额外的空行来创建行，列会自动处理

![[minimal-img-grid-demo.gif]]

## 已知限制

### 仅阅读模式

图片网格仅在阅读视图中可用。图片网格在实时预览或 PDF 导出中不可用。为了获得最佳用户体验，建议通过源码模式而非实时预览来编辑图片网格。

### 发布支持

图片网格兼容 [[Minimal for Obsidian Publish 中文]]，但它们使用了较新的 CSS `:has()` 功能，该功能在旧版浏览器中不可用。

## 快捷键

- 全局切换图片网格
- 在图片块宽度选项之间循环切换

## 辅助类

使用 [[辅助类]] 控制笔记上的图片网格

| 类名       | 说明                      |
| ---------- | ------------------------- |
| `img-grid` | 将图片布局设置为网格模式  |
| `img-100`  | 图片块填充面板宽度的 100% |
| `img-max`  | 图片块填充最大行宽        |
| `img-wide` | 图片块填充宽行宽          |

## 设置

附加选项可以通过 [[Style Settings]] 插件在 **图片** 部分访问：

**图片网格裁剪** 可以选择 _裁剪适配_（填充每个网格单元格的尺寸）或 _显示完整图片_（关闭裁剪并将图片缩小以完全适应网格单元格）。

**图片网格背景** 可以设置为颜色，这在关闭裁剪时想要保留瀑布流外观时很有用。
