## 概述

使用 Minimal 你可以将 [[Dataview]] 表格或无序列表转换为卡片。要使用卡片，请将以下 [[辅助类|辅助类]] 之一添加到 `cssclasses` 属性中：

- `cards` 将笔记中的所有 Dataview 表格转换为卡片
- `list-cards` 将笔记中的所有无序列表转换为卡片

## 辅助类

要使用卡片，请在笔记上添加 `cards` 或 `list-cards` [[辅助类|辅助类]]。因为卡片在幕后使用 [[表格]]，它们也兼容 [[块宽度]] 的表格选项。

以下辅助类可用于进一步自定义特定笔记中的卡片。这些类可以组合使用，但 `cards` 或 `list-cards` 必须存在，其他类才能生效。

| 类名                  | 说明                                    |
| --------------------- | --------------------------------------- |
| `cards`               | 将所有 Dataview 表格设置为卡片布局      |
| `list-cards`          | 将所有无序列表设置为卡片布局            |
| `table-100`           | 卡片块填充面板宽度的 100%               |
| `table-max`           | 卡片块填充最大行宽                      |
| `table-wide`          | 卡片块填充宽行宽                        |
| `cards-align-bottom`  | 将卡片的最后一个元素对齐到底部          |
| `cards-cover`         | 图片调整大小以填充定义的空间            |
| `cards-16-9`          | 将卡片中的图片适配为 16:9 比例          |
| `cards-1-1`           | 将卡片中的图片适配为 1:1 比例（正方形） |
| `cards-2-1`           | 将卡片中的图片适配为 2:1 比例           |
| `cards-2-3`           | 将卡片中的图片适配为 2:3 比例           |
| `cards-cols-1` 到 `8` | 强制指定列数（从 1 到 8）               |

## Style Settings

使用 [[Style Settings]] 你还可以调整卡片的以下设置：

- 最小宽度
- 最大宽度
- 移动端最小宽度
- 内边距
- 最大图片高度
- 边框宽度
- 背景颜色
