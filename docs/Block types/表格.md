#blocks

表格可以使用 Markdown 创建，或通过 [[Dataview]] 自动生成。Minimal 中的表格遵循你的 [[块宽度]] 设置。

## Dataview 设置

使用 [[Minimal Theme Settings]] 定义以下设置：

- **裁剪 Dataview 列** 禁用 [[Dataview]] 单元格中的文字换行，等同于全局为 Dataview 设置 `table-nowrap`
- **Dataview 最大列宽** 设置裁剪前的宽度

## 快捷键

你可以使用 [[快捷键]] 全局调整表格：

- 在全局表格宽度选项之间循环切换

## Style Settings

使用 [[Style Settings]] 插件在全局级别控制表格样式

## 辅助类

使用 [[辅助类]] 在单个笔记级别控制表格样式

| 类名            | 说明                         |
| --------------- | ---------------------------- |
| `table-100`     | 表格块填充面板宽度的 100%    |
| `table-max`     | 表格块填充最大行宽           |
| `table-wide`    | 表格块填充宽行宽             |
| `table-nowrap`  | 禁用表格单元格中的自动换行   |
| `table-wrap`    | 强制表格单元格换行           |
| `table-center`  | 居中窄于行宽的小表格         |
| `table-numbers` | 为表格添加行号               |
| `table-tabular` | 在表格中使用等宽数字         |
| `table-small`   | 在表格中使用小字体           |
| `table-tiny`    | 在表格中使用极小字体         |
| `table-lines`   | 在所有表格单元格周围添加边框 |
| `row-lines`     | 在表格行之间添加边框         |
| `col-lines`     | 在表格列之间添加边框         |
| `row-alt`       | 为交替表格行添加斑马纹背景   |
| `col-alt`       | 为交替表格列添加斑马纹背景   |
| `row-hover`     | 悬停时高亮行                 |
