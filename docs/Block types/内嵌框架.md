#blocks

内嵌框架允许你在笔记中嵌入网页和动态 Web 内容。例如，一个流行的用例是嵌入 YouTube 视频。

内嵌框架兼容 [[块宽度]] 功能。

## 快捷键

使用 [[快捷键]] 全局调整内嵌框架：

- 在内嵌框架宽度选项之间循环切换

## 辅助类

使用 [[辅助类]] 控制特定笔记上的内嵌框架外观：

| 类名          | 说明                        |
| ------------- | --------------------------- |
| `iframe-100`  | 内嵌框架填充面板宽度的 100% |
| `iframe-max`  | 内嵌框架填充最大行宽        |
| `iframe-wide` | 内嵌框架填充宽行宽          |
