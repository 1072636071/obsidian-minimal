#blocks

地图可以使用 [Bases 地图视图](https://obsidian.md/help/bases/views/map) 创建。

地图兼容 [[块宽度]] 功能。

## 辅助类

使用 [[辅助类]] 控制特定笔记上的地图外观：

| 类名       | 说明                    |
| ---------- | ----------------------- |
| `map-100`  | 地图填充面板宽度的 100% |
| `map-max`  | 地图填充最大行宽        |
| `map-wide` | 地图填充宽行宽          |
