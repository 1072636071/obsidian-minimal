#blocks

## 图片相关功能

[[Minimal Theme Settings]] 中的设置：

- [[块宽度]] 允许你控制笔记中图片块的宽度
- [[图片网格]] 允许多列图片布局
- **最大化媒体** 允许图片始终填充行宽

[[Style Settings]] 中的设置：

- [[图片缩放]] 允许你点击并按住图片进行缩放

## 图片滤镜

可以添加图片滤镜来创建以下效果。在图片链接末尾添加滤镜名称使用，例如 `![[image.jpeg#invert]]`

| 滤镜       | 说明                                                    |
| ---------- | ------------------------------------------------------- |
| `#blend`   | 将图片混合到背景中                                      |
| `#circle`  | 将图片裁剪为圆形                                        |
| `#invert`  | 在深色模式下反转图片 — 适用于浅色背景上的图表和手写内容 |
| `#invertW` | 在浅色模式下反转图片 — 适用于深色背景上的图表和手写内容 |
| `#outline` | 在图片周围添加轮廓                                      |

## 快捷键

使用 [[快捷键]] 全局调整图片：

- 全局切换 [[图片网格]]
- 在图片宽度选项之间循环切换

## 辅助类

使用 [[辅助类]] 控制笔记上的图片网格

| 类名        | 说明                       |
| ----------- | -------------------------- |
| `img-blend` | 将所有图片设置为与背景混合 |
| `img-grid`  | 将图片布局设置为网格模式   |
| `img-100`   | 图片块填充面板宽度的 100%  |
| `img-max`   | 图片块填充最大行宽         |
| `img-wide`  | 图片块填充宽行宽           |
