#features

使用 [[Minimal Theme Settings]] 你可以定义以下偏好设置，这些设置与 [[块宽度]] 结合使用非常有用：

- **正常行宽：** 普通文本行的宽度（默认 40em）
- **宽行宽：** "宽"元素的宽度（默认 50em）
- **最大行宽：** 内容在面板内可填充的最大宽度百分比（默认 88%）

要使用这些功能，你需要在 Obsidian 的编辑器设置中开启"可读行宽"。

## 开发者

你可以使用 Minimal Theme Settings 中的 `Dev — Show block widths` 命令来可视化行宽设置。
