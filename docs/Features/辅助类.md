## 如何使用辅助类

辅助类通过特殊的 `cssclasses` [属性](https://help.obsidian.md/Editing+and+formatting/Properties)添加。在源码模式下应该如下所示：

```
---
cssclasses:
  - img-grid
---
```

要使用多个类，请使用以下语法：

```
---
cssclasses:
  - cards
  - cards-1-1
---
```

## 图片网格

[[图片网格]] 功能的辅助类

| 类名       | 说明                                                       |
| ---------- | ---------------------------------------------------------- |
| `img-grid` | 将连续图片转换为列。要创建新行，在图片之间添加额外的空行。 |

## 块宽度

在逐文件基础上启用 [[块宽度]] 功能，用于 [[表格]]、[[图片网格]] 和 [[内嵌框架]]

| 类名                                    | 说明                      |
| --------------------------------------- | ------------------------- |
| `wide`                                  | 整个笔记使用宽行宽        |
| `max`                                   | 整个笔记使用最大行宽      |
| `table-wide`, `img-wide`, `iframe-wide` | 块类型使用宽行宽          |
| `table-max`, `img-max`, `iframe-max`    | 块类型使用最大行宽        |
| `table-100`, `img-100`, `iframe-100`    | 块类型使用面板宽度的 100% |

## 嵌入和内容包含

控制 [[嵌入]] 的样式

| 类名           | 说明                         |
| -------------- | ---------------------------- |
| `embed-strict` | 内容包含无缝地出现在文本流中 |

## 表格

控制 [[表格]] 的行和列样式

| 类名            | 说明                         |
| --------------- | ---------------------------- |
| `table-nowrap`  | 禁用表格单元格中的自动换行   |
| `table-wrap`    | 强制表格单元格换行           |
| `table-center`  | 居中窄于行宽的小表格         |
| `table-numbers` | 为表格添加行号               |
| `table-tabular` | 在表格中使用等宽数字         |
| `table-small`   | 在表格中使用小字体           |
| `table-tiny`    | 在表格中使用极小字体         |
| `table-lines`   | 在所有表格单元格周围添加边框 |
| `row-lines`     | 在表格行之间添加边框         |
| `col-lines`     | 在表格列之间添加边框         |
| `row-alt`       | 为交替表格行添加斑马纹背景   |
| `col-alt`       | 为交替表格列添加斑马纹背景   |
| `row-highlight` | 悬停时高亮行                 |

## 卡片

[[卡片]] 功能的辅助类

| 类名                  | 说明                                    |
| --------------------- | --------------------------------------- |
| `cards`（必需）       | 将所有 Dataview 表格设置为卡片布局      |
| `list-cards`          | 将所有无序列表设置为卡片布局            |
| `cards-align-bottom`  | 将卡片的最后一个元素对齐到底部          |
| `cards-cover`         | 图片调整大小以填充定义的空间            |
| `cards-16-9`          | 将卡片中的图片适配为 16:9 比例          |
| `cards-1-1`           | 将卡片中的图片适配为 1:1 比例（正方形） |
| `cards-2-1`           | 将卡片中的图片适配为 2:1 比例           |
| `cards-2-3`           | 将卡片中的图片适配为 2:3 比例           |
| `cards-cols-1` 到 `8` | 强制指定列数（从 1 到 8）               |
