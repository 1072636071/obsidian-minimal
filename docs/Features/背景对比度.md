#features

## 概述

背景对比度可以通过 [[Minimal Theme Settings]] 更改，并与 [[配色方案]] 兼容。它们也可以通过命令面板和 [[快捷键]] 切换。

浅色模式选项：

- **默认** 为侧边栏使用略深的背景
- **低对比度** 侧边栏和主面板使用相同的背景
- **全白（仅浅色模式）** 侧边栏和主面板使用白色背景
- **高对比度（仅浅色模式）** 侧边栏使用深色背景，主面板使用浅色背景

深色模式选项：

- **默认** 为侧边栏使用略深的背景
- **低对比度** 侧边栏和主面板使用相同的背景
- **纯黑（仅深色模式）** 侧边栏和主面板使用纯黑背景

## 快捷键

- 在浅色和深色模式之间切换
- 在浅色模式背景对比度样式之间循环切换
- 在深色模式背景对比度样式之间循环切换
