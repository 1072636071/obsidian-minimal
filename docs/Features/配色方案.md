#settings

Minimal 通过 [[Minimal Theme Settings]] 包含多种流行的配色方案，并能够通过 [[Style Settings]] 轻松创建你自己的配色方案。

![[minimal-color-schemes.png]]

## 预设配色方案

你可以使用 [[Minimal Theme Settings]] 插件为浅色和深色模式分别定义配色方案。配色方案也可以通过 Obsidian 命令面板或 [[快捷键]] 切换。预设配色方案包括：

- Atom
- Ayu
- Catppuccin
- Dracula
- [[墨水屏模式|墨水屏（测试版）]]
- [Flexoki](https://stephango.com/flexoki)
- Everforest
- Gruvbox
- macOS
- Nord
- Rosé Pine
- Sky
- Solarized
- Things

![[minimal-color-schemes.gif]]

## 自定义配色方案

你可以使用 [[Style Settings]] 插件自定义你的配色方案。

为了更方便地查看更改，可以使用命令面板（默认 `CMD + P`）在新面板中打开 Style Settings。使用 Style Settings 你可以自定义 Minimal 的颜色、字体大小、字体样式以及更多细节。

![[minimal-style-settings.gif]]

## 高级配色方案

如果你熟悉编写 CSS，你可以轻松创建自己的配色方案 [[CSS 代码片段]]。以 [Nord 配色方案 CSS](https://github.com/kepano/obsidian-minimal/blob/master/src/scss/color-schemes/nord.scss) 的结构为例。有关每个 CSS 变量作用的参考，请参见 [[Style Settings]]。
