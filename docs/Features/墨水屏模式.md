#features

墨水屏模式是通过 [[Minimal Theme Settings]] 7.2 提供的实验性 [[配色方案|配色方案]]。

墨水屏模式专为单色显示器设计。它优先考虑高对比度、低抖动和原本依赖颜色的元素的可读性。
