#features

## 概述

块宽度允许你按块类型定义宽度。[[表格]]、[[图片]] 和 [[内嵌框架]] 可以超出正常的文本 [[行宽|行宽]]，在面板中填充更多空间。

块宽度需要在 Obsidian 设置 → 编辑器中开启"可读行宽"。

你可以按块类型和按笔记选择宽度选项。将来 [[Dataview]] 可能会添加按表格设置的功能。（参见 [Dataview issue #675](https://github.com/blacksmithgu/obsidian-dataview/issues/675)）

块布局还使在移动设备上浏览宽表格变得更加容易。表格独立于正文文本水平滚动，因此不会导致整个笔记偏移，也不会将表格内容挤压成不可读的窄列。

## 行宽设置

以下 [[行宽]] 偏好可以通过 [[Minimal Theme Settings]] 设置：

- **正常行宽：** 普通文本行的宽度（默认 40em）
- **宽行宽：** "宽"元素的宽度（默认 50em）
- **最大行宽：** 内容在面板内可填充的最大宽度百分比（默认 88%）

## 辅助类

以下 [[辅助类]] 可用于强制特定笔记中块的宽度：

| 类名                                    | 说明                      |
| --------------------------------------- | ------------------------- |
| `wide`                                  | 整个笔记使用宽行宽        |
| `max`                                   | 整个笔记使用最大行宽      |
| `table-wide`, `img-wide`, `iframe-wide` | 块类型使用宽行宽          |
| `table-max`, `img-max`, `iframe-max`    | 块类型使用最大行宽        |
| `table-100`, `img-100`, `iframe-100`    | 块类型使用面板宽度的 100% |

## 快捷键

- 在图片宽度选项之间循环切换
- 在表格宽度选项之间循环切换
- 在内嵌框架宽度选项之间循环切换

## 开发者

你可以使用命令面板（`CMD + P`）并开启 `Minimal Theme Settings: Show block widths` 开关来可视化块宽度设置。
