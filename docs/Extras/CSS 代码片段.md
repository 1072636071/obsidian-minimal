## 什么是代码片段？

代码片段是 CSS 文件，允许你在 Minimal（或任何 Obsidian 主题）之上创建自己的个人自定义。你可以通过将 CSS 文件添加到仓库中的 `.obsidian/snippets` 文件夹来安装代码片段，并在 Obsidian 中通过 `设置` → `外观` 激活它们。

请注意，代码片段仅推荐给熟悉 CSS 的高级用户。Minimal 的大多数常见功能可以通过 [[Minimal Theme Settings]] 和 [[Style Settings]] 插件更轻松地自定义。[[帮助#与社区 CSS 代码片段的兼容性|阅读更多关于代码片段的帮助和风险。]]
