## 原生 macOS 外观推荐设置

Obsidian 外观设置：

- **原生菜单** 开启
- **窗口边框样式** 设为隐藏

[[Hider]] 插件设置：

- **隐藏应用功能区** 开启
- **隐藏仓库名称** 开启

[[Minimal Theme Settings]] 插件设置：

- **[[导航标签]]** 开启
- **[[配色方案|配色方案]]** 设为 macOS
- **彩色活动状态** 开启

[[Style Settings]]：

- [[标签页样式]] 设为 Modern
