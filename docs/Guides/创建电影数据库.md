#guides

在本指南中，我们将在 Obsidian 中创建一个个人电影数据库。你可以轻松地将电影保存到观影列表，并为看过的电影添加评分。

本指南将涵盖以下步骤：

1. 安装 Obsidian Web Clipper。
2. 添加你的电影模板。
3. 保存电影到你的数据库。
4. 创建视图查看已保存的电影。

## 第 1 步：安装 Web Clipper

为你的浏览器安装 [Obsidian Web Clipper](https://obsidian.md/clipper)。它适用于所有支持扩展的主流浏览器，包括 iOS 上的 Safari 和 Android 上的 Firefox Mobile。

## 第 2 步：在 Web Clipper 中添加电影模板

根据你偏好的电影数据库，下载我现有的 Web Clipper 模板之一：

- [Letterboxd](https://github.com/kepano/clipper-templates/blob/main/templates/letterboxd-clipper.json)
- [IMDB](https://github.com/kepano/clipper-templates/blob/main/templates/imdb-clipper.json)
- [IMDB（引用视图）](https://github.com/kepano/clipper-templates/blob/main/templates/imdb-reference-clipper.json)
- 或者，[创建你自己的模板](https://help.obsidian.md/web-clipper/templates)

导入模板：

1. 打开 Web Clipper 扩展并点击 **设置** 齿轮图标。
2. 转到列表中的任意模板。
3. 点击右上角的 **导入**，或将 `.json` 模板文件拖放到模板区域的任意位置。

## 第 3 步：保存电影

试运行你的 Web Clipper 模板：

1. 在浏览器中打开一个电影页面（如 Letterboxd、IMDB 或其他）。
2. 点击 Web Clipper 扩展。
3. 点击 **添加到 Obsidian**。

你应该会看到电影已作为笔记添加到 Obsidian 中。

## 第 4 步：添加电影数据库

要在一个地方查看所有保存的电影，在 Obsidian 中[创建一个数据库](https://help.obsidian.md/bases/create-base)，命名为 Movies。你可以下载我的 [Movies.base](https://github.com/kepano/kepano-obsidian/blob/main/Templates/Bases/Movies.base) 模板。

你的电影数据库应该有一个过滤器，类似于 `contains(property.categories, "[[Movies]]")`

完成！✨
